# nodemcu-incubator
Fork https://github.com/adriantomic/nodemcu-incubator
NodeMCU/Arduino project for an internet controlled egg incubator.

## Electronics
* DHT22 Temperature/Humidity sensor
* NodeMCU
* Servo motor + controller
* Computer fan
* Relay-module
